## 🚀 Getting Started

### 1. Clone the repository
```bash
git clone https://github.com/ZahidJamal-web/Personal-Portfolio.git
```

### 2. Open project folder
```bash
cd portfolio
```

### 3. Run the project
Simply open:
```text
index.html
```
in your browser.

No additional setup required.
